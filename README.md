# The Grammar of Things

Standalone site for The Grammar of Things (thegrammarofthings.com).

**This repo is independent.** It does not share code, tokens, layouts, or dependencies with the Artists Are Jerks repo.

## Stack

- Astro (SSR, server output mode)
- @astrojs/cloudflare adapter
- Deploys to Cloudflare Pages

## Local development

```sh
npm install
npm run dev       # localhost:4321
npm run build     # production build → ./dist
```
